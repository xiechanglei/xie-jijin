# 基金实时数据查询与持仓盈亏计算工具

支持添加基金、设置持仓金额、实时盈亏计算、份额管理等功能的基金查询工具。

## 安装

全局安装
```bash
npm install -g xie-jijin
# or yarn
yarn global add xie-jijin
```

卸载
```bash
npm uninstall -g xie-jijin
# or yarn
yarn global remove xie-jijin
```

## 使用方法

### 查询基金信息
```bash
jijin
```

### 添加基金到自选
```bash
# 添加基金（不设置持仓金额）
jijin --add 018993

# 添加基金并设置持仓金额（会根据当前基准净值自动计算份额）
jijin --add 018993 --money 10000
```

### 删除自选基金
```bash
jijin --remove 018993
```

### 设置基金持仓金额
```bash
# 为已存在的基金设置持仓金额（会根据当前基准净值重新计算份额）
jijin --set 018993 --money 15000
```

## 功能特点

- **实时数据查询**：获取基金的实时净值、涨跌幅等信息
- **持仓盈亏计算**：根据设置的持仓金额自动计算盈亏
- **份额管理**：自动根据持仓金额和基准净值计算持有份额
- **进度提示**：查询多个基金时显示获取进度
- **数据汇总**：显示总盈亏金额
- **排序显示**：按持有份额倒序排列显示
- **错误处理**：网络请求失败时给出友好提示

## 命令参数说明

- `--add <code>`: 添加基金代码
- `--money <amount>`: 设置持仓金额
- `--remove <code>`: 移除基金代码
- `--set <code>`: 设置指定基金的持仓金额

## 注意事项

- 使用 `--money` 参数时，需要能够获取到基金的基准净值，否则会提示错误
- 持仓盈亏金额 = (当前净值 - 基准净值) × 持有份额
- 基金数据来源于新浪财经，更新时间可能有延迟，仅供参考，不作为投资依据
